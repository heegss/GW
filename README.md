# AuroPortfolio — Android APK

Приложение за следене на портфейл от злато.
Взема цени директно от igold.bg и tavex.bg — **без API ключ, без регистрация**.

---

## Как да получиш APK — БЕЗ Android Studio

### 1. Регистрирай се в GitHub (безплатно)
https://github.com/signup

### 2. Създай ново repository
- Кликни **"+"** горе вдясно → **New repository**
- Ime: `auro-portfolio` (или каквото искаш)
- **Public** (задължително за безплатен build)
- Кликни **Create repository**

### 3. Качи файловете
- Кликни **"uploading an existing file"**
- Плъзни ВСИЧКИ файлове и папки от този ZIP (запази структурата)
- Кликни **Commit changes**

### 4. Изчакай build-а (~3 минути)
- Отиди в **Actions** таба
- Ще видиш "Build APK" — изчакай да стане зелено ✅

### 5. Свали APK-то
- Кликни на завършения build
- Най-долу: **Artifacts** → **AuroPortfolio-APK** → свали ZIP
- Разархивирай → `app-debug.apk`

### 6. Инсталирай на телефона
- Прехвърли APK на телефона (Telegram към себе си / USB)
- Настройки → Сигурност → **"Инсталиране от непознати източници"** → Разреши
- Отвори APK → Инсталирай

---

## Как работи приложението

### Добавяне на продукт:
1. Натисни **+**
2. Paste URL от igold.bg или tavex.bg (напр. `https://igold.bg/zlaten-kanadski-klenov-list-1-oz`)
3. Натисни **Провери** — вижда се цена "Продава" и цена "Купува"
4. Натисни **✓ Използвай тези цени**
5. Въведи колко си платил и количество
6. **Добави в портфейла**

### Печалбата се смята по цена "Купува" ★
Защото това е реалната сума, която ще получиш при продажба на дилъра.

### Обновяване:
- Бутон **⟳** на карта — обновява само този продукт
- Бутон **🔄 Обнови** долу — обновява всички

---

## Поверителност
- Всички данни са САМО на телефона (localStorage)
- Приложението fetch-ва само igold.bg и tavex.bg за цени
- Нищо не се изпраща никъде другаде
